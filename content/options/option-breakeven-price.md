# 期权盈亏平衡价怎么算？

期权盈亏平衡价，是到期时让总损益为零的标的价格。

常见公式：

1. 买入 Call：行权价 + 支付权利金。
2. 买入 Put：行权价 - 支付权利金。
3. Covered Call：正股成本 - 收到权利金。
4. 信用价差：短腿行权价加减净收入。

但新手要注意：盈亏平衡价主要描述“到期结果”，不能机械套用到到期前。

到期前，期权价格还会受时间价值、隐含波动率、买卖价差和成交成本影响。标的价格越过盈亏平衡价，也不代表你当前一定盈利。

下单前更实用的做法，是同时看目标日期的风险图，而不是只看到期曲线。
