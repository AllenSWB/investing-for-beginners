# 期权仓位怎么按压力损失计算？

期权仓位不应该按“权利金便宜”来决定，而应该按压力情景下可能亏多少钱来决定。

基本公式：

`单笔风险预算 = 账户净值 × 单笔允许损失比例`

`合约数 = 向下取整(单笔风险预算 ÷ 每组压力损失)`

实操时建立三层限制：

1. 单笔交易风险不超过账户预设比例。
2. 同一标的所有期权合计风险不超标。
3. 同一主题或高度相关股票的风险合并计算。

有限风险策略可以用最大损失计算，但要预留手续费和滑点。

非有限风险策略不能只看券商保证金，因为保证金不是你的最大亏损。
