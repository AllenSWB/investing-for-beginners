# 期权是什么？Call 与 Put 入门

期权（Option）是一份在约定时间、按约定价格买卖标的资产的权利。期权买方支付权利金获得选择权；期权卖方收取权利金，同时承担合约规定的履约义务。

完整词条：[什么是期权？Call 与 Put 入门](https://xiaoyinsi.com/wiki/options/calls-and-puts)

## Call 和 Put 的区别

- **Call（看涨期权）**：买方有权按行权价买入标的，通常用于表达看涨观点。
- **Put（看跌期权）**：买方有权按行权价卖出标的，可以用于看跌交易或保护已有持仓。

买方可以选择不行权，最大损失通常是已支付的权利金。卖方承担履约义务，部分裸卖策略的风险可能很大。

## 看懂期权要先看什么？

一张期权至少包含：

1. 标的：对应哪只股票、ETF 或指数；
2. 行权价：合约约定的买卖价格；
3. 到期日：这项权利何时失效；
4. Call 或 Put；
5. 买入或卖出方向。

美股标准股票期权通常一张对应 100 股。软件显示的期权报价一般是每股价格，因此报价 2.50 美元的合约通常需要 `2.50 × 100 = 250 美元`。

## 为什么方向看对仍可能亏损？

期权价格除了受标的涨跌影响，还受到剩余时间、隐含波动率、利率、分红和交易流动性影响。方向变化太小、发生得太晚，或者买入时波动率过高，都可能让交易结果低于预期。
