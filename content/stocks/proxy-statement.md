# Proxy Statement 代理声明看什么？

Proxy Statement 是公司在股东大会前披露的治理文件。它包含董事会、投票事项、高管薪酬、股权激励和关联交易等信息。

新手重点看：

1. 高管薪酬指标是否和长期价值挂钩。
2. SBC 和股权激励是否过度。
3. 董事会是否独立。
4. 是否有双重股权、关联交易或特殊控制权。
5. 股东提案和投票建议。

财报告诉你公司赚了多少钱，Proxy 告诉你谁在控制公司、他们怎样拿钱、激励是否对股东友好。
