# 流通股 Float 为什么影响股价波动？

流通股 Float 是市场中实际可自由交易的股份数量。它通常小于总股本，因为内部人持股、受限股份或长期锁定股份暂时不能自由流通。

## Float 与成交量不要混淆

- Float 是可交易股份的总盘子。
- 成交量是某段时间实际换手了多少股份。

低 Float 股票遇到消息和买盘时，价格可能快速上涨；抛售时也可能因买方不足而急跌。可以用“当日成交量 ÷ Float”粗略观察换手强度，但同一股份可能一天内多次成交。

交易低 Float 股票前，应查看买卖价差、停牌风险、内部人解禁和增发可能性，并降低仓位。高波动不是额外收益的保证。
